# Retail AI Inventory Auditor
### Evaluating LLM Accuracy for Enterprise Supply Chain Systems

## Project Overview
This project is an **AI Quality Engineering** framework designed to validate GenAI Agents interacting with retail backends (Manhattan WM/Salesforce). 

## Key Features
* **Data Fidelity:** Ensures AI responses match ground-truth inventory data.
* **Red-Teaming:** Validates that internal SKUs are not leaked to users.
* **Automated Testing:** Built using **PyTest** for CI/CD readiness.

## Tech Stack
* **Python 3.13**
* **PyTest**
